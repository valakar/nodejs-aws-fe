## Setup
To run this project, install it locally using npm:

```
$ npm install
$ npm start
```


To build and deploy application to AWS use:
```
$ npm run cloudfront:update:build:deploy
```

or without confirmation:
```
$ npm run cloudfront:update:build:deploy:nc
```
